## 핵심 결정 & 맥락

### 1. 방향 전환 — 컴패니언 챗봇
- 목표가 "다양한 AI 도구 조합"에서 **"사람 같은 대화 경험"**으로 수렴
- 핵심 가치: 기억, 인격 일관성, 감정적 연결, 몰입감
- 이미지/영상/TTS는 부가 확장으로 후순위

### 2. 인프라 전환 — 로컬 → RunPod
- 8B 로컬 모델로는 "사람 같은" 품질 한계
- 27B~72B 모델 필요 → 로컬 12GB VRAM으로 불가
- RunPod 클라우드 GPU를 메인 인프라로 채택
- 비용 시뮬레이션: 하루 3시간 기준 월 7~18만원

### 3. 모델 후보
- **Gemma 3 27B Abliterated**: Gemini와 가장 가까운 아키텍처. 비용 효율적
- **Qwen 2.5 72B Abliterated**: 한국어 + 감성 대화 최강. 비용 더 높음
- Phase 2에서 직접 비교 테스트 후 확정 예정

### 4. "사람 같은 경험"의 4대 구성 요소
1. 대화 품질 ← 큰 모델
2. 기억 ← 벡터DB + RAG
3. 인격 일관성 ← 페르소나/시스템 프롬프트
4. 몰입감 ← 채팅 UI

### 5. Phase 재구성
- 기존: LLM → 이미지 → 영상 → 평가 → 확장
- 변경: LLM(완료) → RunPod+모델 → 페르소나 → 기억 → UI → 확장
