## 핵심 결정 & 맥락

### 1. 전략 전환 (가장 중요)
- **원래**: 데이터셋 준비 → QLoRA 파인튜닝 → 서빙 (밑바닥 접근)
- **변경**: 이미 잘 만들어진 uncensored 모델을 그대로 실행 → 부족한 부분만 나중에 강화
- **이유**: 사용자가 ML 경험이 없고, 이미 커뮤니티에서 충분히 좋은 uncensored 모델이 존재. 파인튜닝은 별도 ML 학습 프로젝트로 분리.

### 2. 멀티모달 확장
- 원래 텍스트 LLM만 계획 → 이미지 생성/편집, 영상 처리까지 확장
- 이미지: ComfyUI + SDXL/Flux + ControlNet (인체 구조/포즈/표정 강화)
- 영상: 이미지→동영상 변환, 기존 영상 보정 우선 → AI 영상 생성은 나중에

### 3. 인프라
- 로컬 GPU: RTX 3080 Ti (12GB) — 8B LLM, SDXL 이미지 생성 모두 가능
- RunPod는 70B 모델이나 대규모 작업 시에만 투입 (비용 절약)

### 4. ML 학습 프로젝트 분리
- PrivateLLM = 실용 (쓸 수 있는 AI 확보)
- ML 학습 = 이해 (파인튜닝, 모델 원리 배우기) → 별도 프로젝트로 추후 진행
- 추천 경로: Karpathy "Zero to Hero" → GPT-2 파인튜닝 → QLoRA 8B → 심화
